Interactive map of Canadian literary works
An interactive map of the novels written by major Canadian authors, sourced from the Canadian Literary Centre's Annotated Bibliography of Major Canadian Authors, published by UofT. Uses a number of Python textual and geospatial data management methods, as well as coding in HTML, JavaScript, and CSS to create the webmap. Check out the source code in the GitHub repository, and interact with it yourself using MyBinder. Term project by Cameron Brubacher. GEOG 464 - Programming for Geospatial Technologies. Winter 2023. Professor: Emory Shaw. Concordia University.
